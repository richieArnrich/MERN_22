Increment/Decrement Operators

a. Increment Operator (++): Increments the value of a variable by 1.
b. Decrement Operator (--): Decrements the value of a variable by 1.

Types of increment/decrement operators 1. Preincrement / Predecrement operator
2. Postincrement / Postdecrement operator

Working of Preincrement/Preincrement Operators 1. Update the current value by 1
2. Assign the updated value
3. Access the updated value

Working of Postincrement/Postdecrement operators 1. Assign the current value
2. Update the current value by 1
3. Access the updated value
