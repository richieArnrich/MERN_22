Datatypes in JavaScript

1. Datatypes specify what kind of values are stored in the variables
2. Datatypes are used to define the type of data that a variable can hold
3. JavaScript is a dynamically typed language i.e the datatype of the variable depends on the value stored in it
We can store the following types of data in JavaScript variables
a. String
b. Number
c. Boolean
d. Null
e. Undefined
f. Object
